DESIGN THINKING

GESTIÓN Y DOCUMENTACIÓN UX/UI

Es una aplicación para documentar la gestión del Design Thinking.
La gestión se vuelve más fácil de dominar a la hora de llevar

Ver proyecto

Introducción

¡Bienvenidos!

Es una aplicación de gestión del Design Thinking para documentar las fases de: Empatizar - Definir - Idear - Prototipar - Testear, donde en cada una de estas le permita al usuario objetivo (diseñadores) agregar el método/técnica/estrategia que está aplicando y donde pueda subir una foto/imagen y que pueda agregar una descripciones, observaciones y poner fechas de inicio y finalización de cada una.
